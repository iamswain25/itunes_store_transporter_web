iTunes Store Transporter: GUI

Simplify your iTunes workflow.

The Transporter GUI is the easiest way to deliver and manage your content.

Download The Transporter GUI

For Mac, Windows and Linux

Upload with ease

Select your packages with a file system browser, add them to the upload queue, and monitor their progress in real time. No need to juggle multiple terminal windows. You can also set uploads (and all jobs) to run at high or low priority.

All Transporter activity and output are saved

From uploads to metadata retrieval and package status, everything is tracked, stored and easily accessible. Job success/failure, transporter error messages and output, it's all there.

Easily track current and past jobs

Search and sort jobs by date, job type (Transporter mode), status and more.

News

Finally, an Official Release! 2015-01-06

While the iTunes Store Transporter GUI has been relatively stable for some time an official release had yet to be made —until now. The v0.0.1 release contains 2 important additions:

  1. To better support iTMSTransporter options with long paths the length of the transporter_jobs.options column has been extended to 1024 characters
  2. DB indexes have been added to the various columns used for searching
